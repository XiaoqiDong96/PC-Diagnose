# 胰腺癌辅助诊断软件源码

这是 macOS 中文桌面辅助诊断工具的 Python/Tkinter 源码目录。

本目录只保存源码，不包含：

- 打包后的 `.app`
- zip 安装包
- PyInstaller `build/` 或 `dist/` 产物
- 患者级训练数据或结果表

## 启动源码版

将本地训练/性能数据放在 `diagnosis_model.py` 所期待的路径后运行：

```bash
python app_source/diagnosis_app.py
```

## 打包

打包脚本放在仓库 `scripts/` 目录中。打包时需要将脚本和源码放回同一应用开发目录，并准备本地 `data/` 文件夹。

## 模型说明

主诊断结论来自临床、实验室和影像特征的软投票集成模型。技术变量和首次细胞学结果不进入主模型。该工具仅用于临床辅助决策研究，不能替代病理诊断或医生综合判断。
